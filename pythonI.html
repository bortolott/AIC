Programação Básica em Python I

Python é uma linguagem de programação de alto nível, interpretada, procedural e orientada a objetos. É uma linguagem que têm uma incrível quantidade de bibliotecas e frameworks, tornando a linguagem mais atraente para o desenvolvimento de software, além de ser amigável com iniciantes. Segue abaixo os frameworks e bibliotecas mais usadas em 2022.

- Flask

Bibliotecas mais usadas:
- Numpy
- Pandas
- Arrow

Conceitos básicos de programação em Python:

Variáveis: Uma variável não pode ser utilizada em uma expressão sem ter sido inicializada. Não existe “criação automática” de variáveis. Exemplo de atribuição:
reais = euros * taxa

Saída: são os comandos que permitem que dados sejam apresentados no sistema do usuário. Exemplo de saída:
print("Olá mundo!")

Entradas: são os comandos que permitem que dados sejam inseridos no programa, permitindo a manipulação dos mesmos. Exemplo de entrada:
nome = input("Digite o seu nome: ")
print("Olá", nome)

Estruturas condicionais: são estruturas que permitem que um determinado pedaço de código seja executado, caso a condição seja verdadeira.
Dentre as estruturas condicionais temos, if, elif e else.
Demonstração em código:

preço = int(input("Digite o preço do kilo do pão: "))
if preço > 5.00: print("Não comprarei.")
else: print("Comprarei")

Estruturas de repetição: por fim, temos as estruturas de repetição. Que basicamente irão repetir um bloco de código x vezes.
Estruturas de repetição utilizadas em Python: for e while.
